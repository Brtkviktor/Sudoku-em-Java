# 🎮 Sudoku em Java

Projeto desenvolvido durante o curso da DIO com o objetivo de praticar lógica de programação, Programação Orientada a Objetos e manipulação de estruturas em Java através da criação de um jogo de Sudoku interativo no terminal.

## 📚 Sobre o Projeto

O sistema simula um jogo de Sudoku onde o usuário pode iniciar uma partida, inserir números, remover jogadas, validar o estado atual do jogo e finalizar a partida quando todas as posições estiverem preenchidas corretamente.

O projeto foi construído utilizando Java puro, com foco no fortalecimento da lógica de programação, organização de código e aplicação de boas práticas.

---

## 🚀 Funcionalidades

✔️ Iniciar um novo jogo
✔️ Inserir novos números no tabuleiro
✔️ Remover números inseridos pelo jogador
✔️ Visualizar o estado atual do jogo
✔️ Verificar status da partida
✔️ Detectar conflitos e erros no tabuleiro
✔️ Limpar todas as jogadas do usuário
✔️ Finalizar o jogo após validação completa

---

## 🧩 Menu Interativo

O sistema possui um menu interativo com as seguintes opções:

```bash
1 - Iniciar novo jogo
2 - Colocar um novo número
3 - Remover um número
4 - Verificar jogo
5 - Verificar status do jogo
6 - Limpar jogo
7 - Finalizar jogo
8 - Sair
9 - Iniciar Interface Gráfica
```
